# Python Fundamentals Module

Welcome to the **Python Fundamentals** module! This course is designed to guide you through the basics of Python programming, from setting up your environment to more advanced concepts like Object-Oriented Programming.

The module is divided into 5 interactive Jupyter notebooks that you can run directly in Google Colab.

### 2. [Variables, Operators and Data Structures]
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/nextairehorizon/E-Learning-Platform/blob/main/Data%20Science/Module%202%3A%20Python%20Foundations/2%20Variables%20Operators%20Structures.ipynb)

- **Data Structures:** Comprehensive overview of **Lists**, **Tuples**, **Sets**, and **Dictionaries**.

### 3. [Control Flow]
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/nextairehorizon/E-Learning-Platform/blob/main/Data%20Science/Module%202%3A%20Python%20Foundations/3%20Control%20Flow.ipynb)

- **Loops:** Working with `while` and `for` loops, using the `range()` function, and loop control with `break`, `continue`, and `pass`.

### 4. [Functions, Files and Exceptions]
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/nextairehorizon/E-Learning-Platform/blob/main/Data%20Science/Module%202%3A%20Python%20Foundations/4%20Reusability%20and%20Errors.ipynb)

### 5. [Modules and OOP]
